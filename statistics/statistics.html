Example Pluto notebooks on Markov chain Monte Carlo (MCMC) methods

I wrote a few example Pluto notebooks (a ``reactive'' notebook) on MCMC methods. There are many high quality packages to perform MCMC sampling. Nevertheless, I found it very helpful to write a few basic example MCMC codes and apply them to a simple examples, which is what constitutes these notebooks. These notebooks may contain errors. Please let me know if you spot any, or if you would like to improve the notebooks in any way.
